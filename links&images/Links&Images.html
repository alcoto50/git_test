We can have the link open in a new tab using the TARGET attribute. While href specifies the destination Target specifies where the link should be opened. Can also use _BLANK to open the link.

the noopener value of the rail attribute ensures that a link opened in a new window cannot interact or access the origional page.


There are two kinds of links, Absolute, and Relative .

1.Absolute links to other websites on the internet. The syntax scheme for an absolute link is: //domain/path. The absolute domain will always have the scheme and domain.


2. Relative links to other pages within our own website. They do not include domain names since it is directed towards the same page. The only include the file path to the other page.
